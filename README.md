# KHALILSHOTS — Photography Portfolio

A personal photography portfolio website built with vanilla HTML and CSS.

## Pages

- **Home** — artist statement and intro
- **Portfolio / Final** — final photo series with storyboard and moodboard
- **Portfolio / Midterm** — midterm project gallery
- **Best Photos** — curated selection of best work
- **About Me** — bio and artist statement
- **Contact** — social links and CV

## Tech

No frameworks, no dependencies — plain HTML, CSS, and static images.
